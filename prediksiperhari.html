1. Persiapan ‚Äî Konversi Durasi ke Jam Desimal

Data mentah mencatat durasi dalam format H:MM:SS. Sebelum digunakan dalam proses analisis, durasi harus dikonversi ke bentuk jam desimal agar dapat dihitung secara numerik. Nilai jam desimal inilah yang menjadi nilai aktual Y‚Çú untuk setiap record.

Rumus Konversi
Jam Desimal = Jam + (Menit √∑ 60) + (Detik √∑ 3600)
Contoh: Record No. 1 ‚Äî Durasi 0:06:17
= 0 + (6 √∑ 60) + (17 √∑ 3600)
= 0 + 0.1000 + 0.00472
= 0.1047 jam ‚Üí Y‚ÇÅ = 0.104722
Contoh: Record No. 2 ‚Äî Durasi 0:28:10
= 0 + (28 √∑ 60) + (10 √∑ 3600)
= 0 + 0.4667 + 0.00278
= 0.4694 jam ‚Üí Y‚ÇÇ = 0.469444
Catatan Penting:

Karena peramalan dilakukan per-record, tidak ada proses rata-rata bulanan. Setiap durasi pekerjaan langsung menjadi nilai Y‚Çú dalam urutan kronologis. Total terdapat 105 nilai Y‚Çú (Y‚ÇÅ hingga Y‚ÇÅ‚ÇÄ‚ÇÖ) yang mewakili setiap transaksi perbaikan meteran sepanjang tahun 2025.

4.3.2 Evaluasi Akurasi ‚Äî Perhitungan MAPE Per-Record

Setelah memperoleh hasil peramalan Holt-Winters per-record, dilakukan evaluasi akurasi model menggunakan indikator Error, Absolute Error, APE, dan MAPE pada periode data testing (No. 74‚Äì106).

Rumus Perhitungan Akurasi

1. Error
Error = Y‚Çú ‚àí F‚Çú    (selisih aktual dan forecast)
2. Absolute Error
|Error| = |Y‚Çú ‚àí F‚Çú|    (nilai mutlak tanpa tanda)
3. APE (Absolute Percentage Error)
APE = (|Y‚Çú ‚àí F‚Çú| / Y‚Çú) √ó 100%
4. MAPE (Mean Absolute Percentage Error)
MAPE = Œ£ APE √∑ Jumlah Record Testing (n)
// n = 33 record testing (No. 74‚Äì106)
